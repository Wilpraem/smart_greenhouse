# Smart Greenhouse

Программный прототип умной оранжереи для автономного ухода за тропическими растениями.

## Описание проекта

Проект моделирует работу умной оранжереи и автоматически поддерживает условия выращивания для нескольких растений.

Система:
- контролирует температуру;
- контролирует влажность почвы;
- контролирует уровень освещённости;
- прогнозирует цветение и плодоношение;
- использует ИИ через NVIDIA API;
- автоматически переходит в резервный режим при сбое связи.

## Основные возможности

- индивидуальный контроль для каждого растения;
- работа с 4 растениями:
  - орхидея фаленопсис;
  - антуриум;
  - бостонский папоротник;
  - банан;
- ручное моделирование аварий:
  - жара;
  - сухая почва;
  - нехватка света;
  - перелив;
  - отключение интернета;
- встроенный HTTP API:
  - `/health`
  - `/snapshot`

## Структура проекта

- `models.py` — модели данных и параметры растений
- `simulator.py` — симуляция среды и логика изменения состояния
- `ai_controller.py` — управление через NVIDIA API
- `fallback_controller.py` — резервный автономный режим
- `app.py` — основной терминальный интерфейс
- `api_server.py` — HTTP API для выдачи состояния системы
- `gui.py` — графическая версия интерфейса

## Как работает проект

1. Создаётся модель оранжереи и растений.
2. Симулятор обновляет температуру, влажность и освещённость.
3. В обычном режиме решение принимает ИИ через NVIDIA API.
4. Если внешний сервис недоступен, включается резервный режим.
5. События и состояние растений выводятся в интерфейс и доступны через `/snapshot`.
